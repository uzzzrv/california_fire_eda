Проект: Анализ и прогнозирование серьезных пожаров в Калифорнии

Цель проекта: предсказать, будет ли пожар классифицирован как MajorIncident (серьезное происшествие), на основе имеющихся характеристик.

Данные:
- 1636 записей о пожарах в Калифорнии (2013-2019 годы)
- 40 исходных признаков
- Целевая переменная: MajorIncident (24 процента серьезных пожаров, 76 процентов обычных)

Этапы работы:
1. EDA - анализ пропусков, выбросов, типов данных
2. Предобработка - удаление признаков с более чем 80 процентов пропусков, нормализация, One-Hot Encoding
3. Корреляционный анализ - удаление мультиколлинеарных признаков
4. PCA - снижение размерности (44 компоненты для 90 процентов дисперсии)
5. t-SNE - визуализация структуры данных
6. Моделирование - Logistic Regression, Random Forest, Gradient Boosting
7. Анализ важности признаков

Результаты:

Лучшая точность: 85.13 процента (Random Forest и Gradient Boosting)

Топ-5 важнейших признаков (Random Forest):

1. AcresBurned (площадь пожара) - 20.49 процента
2. Latitude (широта) - 12.45 процента
3. Longitude (долгота) - 12.15 процента
4. ArchiveYear (год) - 7.76 процента
5. CalFireIncident - 5.34 процента

Выводы:

Площадь пожара является главным фактором определения серьезности происшествия. Географическое положение также сильно влияет на риск. Наблюдается тренд роста серьезности пожаров по годам. Модель с точностью 85 процентов может использоваться для распределения ресурсов пожарных служб.
